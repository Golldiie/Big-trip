# Big trip

Сервис для планирования путешествий

## Стек

- HTML
- CSS
- JavaScript
- Webpack

## Возможности

- загрузка точек маршрута с сервера;
- создание, редактирование и удаление точек маршрута;
- фильтрация маршрутов
- сортировка по дате, длительности и стоимости;
- динамическое отображение информации о направлениях и дополнительных опциях;
- взаимодействие с REST API;
- работа с датами через Flatpickr;
- отображение состояний загрузки и обработки запросов;
- обработка ошибок сервера;

## Запуск

npm install
npm start
